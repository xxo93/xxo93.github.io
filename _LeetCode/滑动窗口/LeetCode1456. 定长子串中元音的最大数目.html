算法解析

为什么用滑动窗口？

暴力做法：对每个起点都重新数一遍元音，O(n·k)。但相邻两个窗口只差一个字符——左边移出一个、右边移入一个。所以只要做一减一加，O(n) 搞定。

具体步骤

  • 先数出前 k 个字符里的元音数，作为初始值。
  • 窗口每次右移一格：移出的字符是元音就 -1，移入的字符是元音就 +1。
  • 每滑动一次，更新最大值。
  • 遍历结束，返回最大值。

代码

class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        vowels = set('aeiou')
        count = 0

        # 初始化第一个窗口
        for i in range(k):
            if s[i] in vowels:
                count += 1
        max_count = count

        # 滑动窗口：每次右移一格
        for i in range(k, len(s)):
            if s[i - k] in vowels:   # 移出左端
                count -= 1
            if s[i] in vowels:       # 移入右端
                count += 1
            max_count = max(max_count, count)

        return max_count

复杂度

时间：O(n) — 每个字符最多被访问两次（加入和移出各一次）。

空间：O(1) — 只用了几个变量。
