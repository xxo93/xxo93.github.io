Z 字形变换

将一个给定字符串 s 根据给定的行数 numRows，以从上往下、从左到右进行 Z 字形排列，然后逐行读取得到新字符串。

📋题意解读

简单来说，这道题在问什么？

想象你有一根绳子（字符串），要把上面的字符按 Z 字形排列成多行：

s = "PAYPALISHIRING", numRows = 3 排列成 Z 字形： P A H N ← 第 0 行 A P L S I I G ← 第 1 行 Y I R ← 第 2 行 然后从上到下、从左到右逐行读取： → "PAHN" + "APLSIIG" + "YIR" = "PAHNAPLSIIGYIR"

就像蛇形走楼梯：先往下走到底，再斜着往上走回来，如此反复。最后把每一行的字符拼起来。

生活类比

想象你在一面墙上按 Z 字形挂照片：

  • 先从最上面一排往下挂
  • 挂到最底下后，斜着往上挂回来
  • 到了最上面再往下挂……
  • 挂完后，按每一行从左到右的顺序读出照片编号

更多示例

示例 2：s = "PAYPALISHIRING"，numRows = 4

P I N ← 第 0 行 A L S I G ← 第 1 行 Y A H R ← 第 2 行 P I ← 第 3 行 结果: "PINALSIGYAHRPI"

🔍为什么用"模拟"而不是其他算法？

为什么不用滑动窗口 / 动态规划 / DFS？

  • 滑动窗口：适合找满足某种条件的连续子数组/子串，但这道题不是在"找"，而是在"排列"
  • 动态规划：适合有重叠子问题的最优化问题，这里没有最优解的概念
  • DFS / 回溯：适合搜索所有可能路径，这里路径是固定的
  • 双指针：适合在有序数据中搜索，这里不涉及查找

这道题的核心是按规则把字符放到对应行，最直接的方法就是模拟这个过程——用一个变量跟踪"当前行"和"方向"，逐字符放置。

核心思路

准备 numRows 个"篮子"（字符串），然后沿着 Z 字形路径走：

  • 每走一步，把当前字符扔进当前行的篮子
  • 走到最底下 → 调头向上
  • 走到最上面 → 调头向下
  • 走完后，把所有篮子的字符按顺序拼接起来

关键：只需要 numRows 个字符串 + 一个方向变量，一次遍历就能完成。

📐算法具体步骤

  1. 边界判断：如果 numRows == 1 或 numRows >= len(s)，直接返回原字符串（不需要 Z 字形排列）。
  2. 准备篮子：创建 numRows 个空字符串 rows，用来存放每一行的字符。
  3. 初始化方向：设当前行 cur_row = 0，方向 direction = 1（1=向下走，-1=向上走）。
  4. 逐字符放置：遍历字符串中的每个字符：
    • 把当前字符追加到 rows[cur_row]
    • 如果 cur_row == 0：到顶了，方向改为向下（direction = 1）
    • 如果 cur_row == numRows - 1：到底了，方向改为向上（direction = -1）
    • cur_row += direction（沿方向移动到下一行）
  5. 拼接结果：把所有行的字符串拼起来 ''.join(rows)，返回。

💻代码实现（Python）

solution.py 
class Solution:
    def convert(self, s: str, numRows: int) -> str:
        # 边界情况：只有一行或行数 ≥ 字符串长度，无需变换
        if numRows == 1 or numRows >= len(s):
            return s

        # 创建 numRows 个空字符串，每行一个"篮子"
        rows = [''] * numRows
        cur_row = 0
        direction = 1  # 1 = 向下, -1 = 向上

        # 遍历每个字符，按 Z 字形路径放入对应行
        for c in s:
            rows[cur_row] += c

            # 到达顶部或底部时调头
            if cur_row == 0:
                direction = 1
            elif cur_row == numRows - 1:
                direction = -1

            cur_row += direction

        # 拼接所有行
        return ''.join(rows)

📊复杂度分析

时间复杂度

O(n)

只遍历了一遍字符串（n 个字符），每个字符处理 O(1)。最后拼接 rows 也是 O(n)。总计 O(n)。

空间复杂度

O(n)

需要 numRows 个字符串来存储所有字符，总字符数为 n，所以额外空间为 O(n)。
